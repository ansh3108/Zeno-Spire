# Zeno Spire

It is an infinite, multi scale platformer game. 

##  Controls

| Action | Key |
|--------|-----|
| **Move** | `A` / `D` |
| **Jump** | `Space` *(Double Jump Enabled)* |
| **Shrink (Micro)** | `Q` |
| **Grow (Macro)** | `E` |
| **Zeno Dash** | `Shift` |
| **Grab / Drop Key** | `F` |

## Mechanics

* Shift(Macro & Micro): You can scale the universe using Q and grow using E, dynamically altering player's size, gravity and movement speed.
* Zeno Dash: Added a traversal mechanic where the player crosses infinite gaps by taking an infinite series of exponentially smaller, rapid fire hops.
* Fractal Key: Collectible that scales with the player, shrink to grab it and grow to turn it into a massive boulder
* Environment: The world transforms depending on your size, shifts from a non lit metal at normal scale to glowing cell when tiny and massive when giant.
* Audio & Atmosphere: It has a procedural web audio synth from scratch that generates crisp sounds for every jump, dash and landing. The atmosphere is tied directly to your scale, with the background particle system naturally transforming from standard floating dust into massive, slow moving boulders when you shrink, and twinkling stars when you grow.

## Theme
It fits the endless theme by using procedural generations to build a tower that never actually runs out of platforms, you cross infinite gaps using infinite, shrinking steps while macro and micro shifts make sure theres no final level(in both directions)
