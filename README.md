# Zeno Spire

It is an infinite, multi scale platformer game. 

## Controls

##  Controls

| Action | Key |
|--------|-----|
| **Move** | `A` / `D` |
| **Jump** | `Space` *(Double Jump Enabled)* |
| **Shrink (Micro)** | `Q` |
| **Grow (Macro)** | `E` |
| **Zeno Dash** | `Shift` |
| **Grab / Drop Key** | `F` |
